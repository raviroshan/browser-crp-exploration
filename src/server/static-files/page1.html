Notes:

  1. CSS is considered a "render blocking resource". This means that the Render Tree cannot be constructed without first fully parsing the resource.
  2. CSS can also be "script blocking". This is because JavaScript files must wait until the CSSOM has been constructed before it can run.
  3. JavaScript is considered a "parser blocking resource". This means that the parsing of the HTML document itself is blocked by JavaScript.
  4. To avoid JavaScript being parser blocking, it can be loaded asynchronously be applying the async attribute.
